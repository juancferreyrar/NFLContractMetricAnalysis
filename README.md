# NFLContractMetricAnalysis
A project that aims to explore the association between key-metrics for each main offensive position in the NFL (QB,RB,WR) on their first season for a team to the value of a players contract on that same season

To explore the report generated for the project (for a non data-analysis background), click on the PDF file, to access the notebook with the data exploration and more data-depth explanations, click on the ipynb file, and finally, to access the datasets extracted from Kaggle, click on the xlsx/csv files

Sources:

Contract Data: https://www.kaggle.com/datasets/nicholasliusontag/nfl-contract-and-draft-data

Player Statistic Data: https://www.kaggle.com/datasets/philiphyde1/nfl-stats-1999-2022/data
